# Optical-Constant-Calculator
The Optical Constant Solver GUI is a Python-based graphical user interface developed to calculate wavelength-dependent differential optical constants, molar optical coefficients, and dissymmetry factors from experimental chiroptical measurements. The software accepts an Excel (.xlsx) file containing wavelength, circular dichroism (CD), circular differential scattering (CDS), total extinction (E), absorption extinction (A), and scattering extinction (S) spectra measured at multiple concentrations. The input columns should be named CD_X, CDS_X, E_X, A_X, and S_X, where X denotes the sample concentration, and the concentration labels must be consistent across all datasets.

After the user enters the instrument calibration constant (K) and optical path length (l), the program calculates the differential absorption (ΔA), differential scattering (ΔS), and differential extinction (ΔE) spectra. Linear regression of each optical quantity versus concentration is then performed at every wavelength to determine the molar optical coefficients.

The GUI displays all calculated spectra and optical parameters as interactive plots and allows the processed results to be exported to an Excel workbook containing separate worksheets for the spectra, molar optical coefficients, and dissymmetry factors.
